# Notes App - React Native Expo

A clean and modern Notes application built using React Native and Expo.

## What I Built

- Create Notes
- Edit Notes
- Search Notes
- Light/Dark Theme Toggle
- Responsive UI for Mobile & Tablet
- Smooth and Modern UI Design

---

# Advanced Concepts Used

- Component Separation Architecture
- Orchestrator Pattern
- Reusable Components
- Dynamic Theming System
- Responsive Layout Handling
- FlatList Optimization
- React Memoization (`useMemo`, `React.memo`)
- Keyboard Safe Editor using `KeyboardAvoidingView`
- Clean Folder Structure
- Scalable State Management Pattern

---

# Tech Stack

- React Native
- Expo
- TypeScript
- React Hooks
- FlatList
- Animated API
